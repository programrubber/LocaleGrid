# LocaleGrid 릴리즈 노트

## 0.8.0 - 2026-06-30

### 변경 사항
- 검색 결과에서 매칭된 텍스트를 선명한 하이라이트로 표시하도록 개선했습니다.

## 0.7.0 - 2026-06-01

### 변경 사항
- 플러그인 로고를 원본 이미지 기반 아이콘으로 교체.

## 0.6.0 - 2026-05-31

### 변경 사항
- 플러그인 상세 설명의 깨진 한국어 문구를 UTF-8 기준으로 복구했습니다.
- 릴리즈 ZIP 내부 `META-INF/plugin.xml`에서 한국어 설명이 정상 출력되는지 확인했습니다.
